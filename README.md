# tckt

**tckt** consists of a client component **tckc** and server component **tcks** built to analyze multi-threaded access to a shared resource.

## Build
`./gradlew bootJar`

## Test
`./gradlew check`

## Usage

### Client
`java -jar tckc.jar`

### Server
`java -jar tcks.jar`
